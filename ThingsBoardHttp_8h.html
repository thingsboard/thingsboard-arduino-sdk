class  ThingsBoardHttpSized< Logger >
 Wrapper around the ArduinoHttpClient or HTTPClient to allow connecting and sending / retrieving data from ThingsBoard over the HTTP orHTTPS protocol. BufferSize of the underlying data buffer as well as the maximum amount of data points that can ever be sent have to defined as template arguments. Changing is only possible if a new instance of this class is created. If theese values should be changeable and dynamic instead. Simply set THINGSBOARD_ENABLE_DYNAMIC to 1, before including ThingsBoardHttp.h. More...
